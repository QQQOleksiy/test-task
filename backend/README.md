# RecipeTypes Book API

Бекенд для застосунку "RecipeTypes Book", який використовує TheMealDB API для надання доступу до інформації про рецепти.

## Технології

- Node.js
- Express.js
- TypeScript
- Axios

## Структура проекту
Базовий Express сервер для веб-додатку.
